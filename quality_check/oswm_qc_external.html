OSM Inspector (Geofabrik)

Geofabrik offers OSM Inspector. A visualization tool containing detections for potential errors in OSM features. We curated some themes that can be correlated to Pedestrian data, but some may include highways, bridges, etc. You can navigate through the following themes:

Theme (link) Description
Highways Theme Detections related to highways, we kept only the ones with unknown tags, like nodes improperly tagged as footways
Geometry Theme Detections related to geometry, many detections pertains roads
Routing Theme Detections related to routing, we kept only duplicated edges (in general) and non car/bike islands. Keep in mind that some 'islands' are meant to be as such.
Tagging Theme Detections related to tagging, can regard any kind of feature
Area Theme Can hold detections about Pedestrian Areas, but most aren't about them
