Data Hub Highlights & Capabilities

Acquisition Hub

  • Exhaustive bulk discovery bypassing Tasking Manager search index limitations
  • Precise geometry-based spatial checks to identify overlapping pedestrian projects
  • Integration with multiple project management endpoints

Watcher Dashboard

  • 120-day rolling history tracking of additions, modifications, and deletions
  • Per-layer updates visualization (sidewalks, crossings, kerbs, footways)
  • RSS feeds for daily updates and changeset vandalism alerts
  • Deep link auditing with OSM and OSMCha

Interactive API

  • Serverless endpoints for processed GeoParquet and PMTiles layers
  • Comprehensive description catalog for every dataset and schema
  • Web-based playground for direct data querying and VRT linking
